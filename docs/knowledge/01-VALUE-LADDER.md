# Value Ladder - Escada de Produtos

## Visão Geral

A Value Ladder NEXORAMA segue o modelo de Russell Brunson: cada degrau aumenta valor entregue, preço e profundidade do relacionamento.

```
                    ┌─────────────┐
                    │   AGÊNCIA   │ R$ 25k-80k
                    │  Done-for-  │
                    │    you      │
                ┌───┴─────────────┴───┐
                │      MENTORIA       │ R$ 4.500
                │   Done-with-you     │
            ┌───┴─────────────────────┴───┐
            │         FRAMEWORK           │ R$ 997
            │        Self-paced           │
        ┌───┴─────────────────────────────┴───┐
        │            NEWSLETTER               │ Grátis
        │         Show, don't tell            │
        └─────────────────────────────────────┘
```

## Nível 1: Newsletter

### Dados
| Campo | Valor |
|-------|-------|
| Nome | Content-to-App Weekly |
| Preço | Grátis |
| Modelo | Show, don't tell |
| Formato | 1 Conteúdo → 1 App por semana |
| Frequência | Semanal (terça ou quinta) |

### Estrutura de Cada Email
1. **Conteúdo original** - O que transformei esta semana
2. **Extração** - O que encontrei de metodologia
3. **App** - O que criei (com link demo)
4. **Bastidores** - Como fiz (processo)
5. **CTA** - Quer isso pro seu conteúdo?

### Métricas
| Métrica | Meta 90 dias |
|---------|--------------|
| Subscribers | 1.000 |
| Open Rate | 40-50% |
| Conversão para Framework | 2-3% |

### Insights das Minds
> **@Seth Godin**: "Newsletter é permission marketing puro. Não escreva SOBRE, MOSTRE acontecendo."

> **@Dan Koe**: "Newsletter é fundação do one-person business. Personal monopoly de José: único que transforma conteúdo em app toda semana."

### Desbloqueio
- **Requisito**: Nenhum (entrada do funil)
- **Desbloqueia**: Framework (com 500 subscribers)

## Nível 3: Mentoria

### Dados
| Campo | Valor |
|-------|-------|
| Nome | Mentoria Content-to-App |
| Subtítulo | Crie 3 Apps em 6 Semanas |
| Preço | R$ 4.500 |
| Formato | Cohort 6 semanas + suporte individual |
| Capacidade | 8 pessoas por turma, 1 turma/mês |
| Receita máxima | R$ 36.000/mês |

### Estrutura das 6 Semanas
| Semana | Foco |
|--------|------|

### Entregável
Mínimo 3 apps funcionais publicados

### Garantia
3 apps ou próxima turma grátis

### Pré-requisito
Ter feito Framework ou aplicação aprovada

### Insights das Minds
> **@Naval Ravikant**: "Venda o que só VOCÊ pode fazer. José vê padrões que outros não veem - isso é specific knowledge."

## Sistema de Desbloqueio

```
Newsletter (ativo)
    │
    └──► 500 subscribers ──► Framework (pronto)
                                   │
                                   └──► 20 vendas ──► Mentoria (pronto)
                                                           │
                                                           └──► 5 formados ──► Agência
```

### Lógica
1. **Newsletter** sempre ativa (entrada)
2. Cada nível tem métricas que desbloqueiam o próximo
3. Métricas são cumulativas (ex: para Agência, precisa de 500 sub + 20 vendas + 5 formados)
