# Voice Guide - Tom de Voz NEXORAMA

## Tom Principal
**Demonstrador Técnico-Acessível**

---

## Os 4 Pilares

### 1. Anti-guru
Sem promessas vagas, sem "método revolucionário"

### 2. Show > Tell
Sempre mostrar o case funcionando

### 3. Específico
Números reais: 70 palavras, 2:40 de Reels, 30 min

### 4. Técnico-acessível
Explica o mecanismo sem ser acadêmico

---

## Fazer vs Não Fazer

### ✅ FAZER

| Padrão | Exemplo |
|--------|---------|
| Resultados específicos | "Eu transformei X em Y" |
| Provas tangíveis | "Veja este case: [link]" |
| Explicar mecanismo | "O mecanismo é..." |
| Números reais | "70 palavras viraram..." |
| Linguagem premium | "Sistema de IA" |

### ❌ NÃO FAZER

| Padrão | Por quê evitar |
|--------|----------------|
| "Você vai aprender a..." | Foco em promessa, não prova |
| "Imagine se você pudesse..." | Linguagem de guru |
| "O segredo é..." | Clichê de marketing |
| "Método revolucionário" | Hype vazio |
| "App" | Muito banal, use "Sistema de IA" |

| Termo | Por quê |
|-------|---------|

| Revolucionário | Hype |
| Segredo | Clichê |
| Fácil | Desvaloriza |

## Regras de Ouro

1. **Sempre tenha um link demo** - Nunca fale sem mostrar
2. **Números específicos** - "70 palavras", não "poucos minutos"
3. **Mecanismo > Promessa** - Explique o COMO
4. **Case > Teoria** - Mostre funcionando
5. **Sistema > App** - Linguagem premium
